# Relay Error Codes

Machine-readable codes are defined on `RelayError` subclasses in the Python package (`relay/errors.py`). Primary codes:

| Code | Class |
|------|--------|
| E001 | ParseError |
| E002 | TypeMismatchError |
| E003 | SchemaNotFoundError |
| E004 | RelayReferenceError |
| E005 | DeltaConflictError |
| E006 | ValidationError |
| E007 | EncodingError |
| E008 | DecodingError |
| E009 | RegistryError |
| E010 | SchemaHashMismatch |
